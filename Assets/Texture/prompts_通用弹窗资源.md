# 通用弹窗 UI 资源生成 Prompt

基于项目 **像素风（Pixel Art）** 规范，以下 prompt 用于生成弹弹堂风格通用弹窗所需资源。生成时请保持**统一色板、像素网格对齐、清晰对比度**，便于在 Unity 中作为 Sprite 使用。

---

## 1. 弹窗底板背景（可九宫格切片）

**用途**：弹窗主面板背景，建议做 9-Slice 以适配不同尺寸。

**Prompt：**
```
像素风弹窗底板背景，400×300 像素，用于九宫格切片 UI。柔和圆角矩形，限色 8～16 色：暖棕/米色底、深棕色描边（2～3 像素轮廓），内侧轻微阴影、上缘一点高光。无渐变，纯块面色、像素边缘清晰。风格：休闲游戏/弹弹堂，干净易读。四边预留约 16 像素安全边供切片用。PNG，面板外透明。
```

**尺寸建议**：400×300 或 512×384，留出四边约 16px 作为九宫格边距。

---

## 2. 关闭按钮（默认态）

**用途**：弹窗右上角关闭按钮，未按下状态。

**Prompt：**
```
像素风关闭按钮图标，64×64 像素，8～16 色。深红或深棕的圆形/圆角方形底，上绘白色或浅灰「X」。2 像素描边/边框，像素网格对齐，无抗锯齿。小尺寸下对比清晰，便于在 Unity 中使用。边缘留 4 像素内边距。PNG，背景透明。
```

**尺寸**：64×64（可再提供 32×32 小尺寸版本）。

---

## 3. 关闭按钮（悬停/按下态）

**用途**：关闭按钮 hover 或 pressed 状态，可与默认态同尺寸做状态切换。

**Prompt：**
```
像素风关闭按钮图标按下/悬停态，64×64 像素，与默认态同风格。背景略加深，「X」向右下偏移 1～2 像素以表现按下感。同套限色与像素网格，4 像素内边距。PNG，背景透明。
```

---

## 4. 确认按钮（主按钮）

**用途**：弹窗内“确定”“确认”等主操作按钮。

**Prompt：**
```
像素风确认/确定按钮，128×48 像素（或 64×64 方形）。主色为绿色或金色，2 像素深色描边，上缘轻微高光（1～2 像素线）。圆角矩形，无渐变，块面色。风格：弹弹堂/休闲 2D 游戏，小尺寸下清晰可读。内边距 4 像素。PNG，按钮外透明。
```

**尺寸**：128×48 横条或 64×64 方钮，按实际布局二选一或各做一份。

---

## 5. 取消按钮（次要按钮）

**用途**：弹窗内“取消”“关闭”等次要操作。

**Prompt：**
```
像素风取消按钮，128×48 像素，与确认按钮同尺寸。主色为灰色或中性棕，2 像素深色描边，上缘高光风格与确认按钮一致。视觉上次于确认按钮，但遵循相同像素风规则。4 像素内边距。PNG，按钮外透明。
```

---

## 6. 弹窗标题栏装饰条

**用途**：弹窗顶部标题区域背景条或装饰带。

**Prompt：**
```
像素风弹窗标题栏装饰条，384×48 像素，8～16 色。横向条带：用 2～3 层块面色带代替渐变（中间深、上缘略亮）。可选两侧简单缎带或缎带端头，仅用像素块表现。线条粗细统一 1～2 像素，网格对齐。PNG，条带上下透明。
```

**尺寸**：宽度略小于底板（如 384），高度约 48px，与底板风格一致。

---

## 7. 弹窗四角装饰（可选）

**用途**：弹窗四角的小装饰，增强弹弹堂/休闲风格。

**Prompt：**
```
像素风弹窗四角装饰，32×32 像素，单角（如左上角）。简单图案：小缎带卷、木纹角撑或小宝石等。色板与弹窗底板一致（棕/金/米色），1～2 像素描边。像素精确，无平滑曲线。PNG，透明。可提供四个角各一或一张可旋转复用。
```

**尺寸**：32×32 单角，需四个角可旋转或做 4 张图。

---

## 8. 通用分割线（横条）

**用途**：弹窗内分隔标题与内容、或内容与按钮区域。

**Prompt：**
```
像素风 UI 横向分割线，256×8 像素。单像素或 2 像素粗的横线，深棕或灰色，可选轻微双线（阴影）效果。直线、网格对齐、无曲线。PNG，上下透明。
```

**尺寸**：256×8（宽度可随内容区拉伸或平铺）。

---

## 9. 弹窗遮罩/半透明蒙版（可选）

**用途**：弹窗后的全屏暗色蒙版，通常用 Unity 色块 + 透明度实现即可；若需贴图可用小图平铺。

**Prompt：**
```
像素风 UI 遮罩用轻微噪点或纹理，64×64 像素，可平铺。极深灰带轻微纹理，4～8 色，低对比度以保持背景感。可选：轻微网格或点阵。PNG，半透明亦可。
```

**说明**：多数项目用纯色 + Alpha 做遮罩，本项仅当需要纹理时使用。

---

## 使用与规范小结

| 资源           | 建议尺寸    | 格式 | 备注                     |
|----------------|-------------|------|--------------------------|
| 弹窗底板       | 400×300+    | PNG  | 九宫格留边约 16px        |
| 关闭按钮       | 64×64       | PNG  | 默认 + 悬停/按下各一     |
| 确认/取消按钮  | 128×48 或 64×64 | PNG  | 与底板色系统一           |
| 标题栏条       | 384×48      | PNG  | 与底板风格一致           |
| 角装饰         | 32×32       | PNG  | 四角或可旋转             |
| 分割线         | 256×8       | PNG  | 可平铺                   |

- **色板**：整批资源尽量共用一套 8～32 色（如棕/米色/深棕/金/绿/灰），保证界面统一。
- **描边**：按钮与底板用 2px 左右深色描边，线条粗细统一。
- **内边距**：所有 Sprite 留 4px 左右安全边，避免图集裁切。
- **风格**：弹弹堂/休闲 2D、像素块明确、避免写实材质与复杂渐变。

以上 prompt 可直接用于 AI 绘图或交给美术按规范出图；若引擎内已有主题色变量，可在 prompt 中替换为具体色值描述。
