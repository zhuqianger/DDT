# 通用弹窗 UI 资源生成 Prompt

基于项目 **像素风（Pixel Art）** 规范，以下 prompt 用于生成弹弹堂风格通用弹窗所需资源。生成时请保持**统一色板、像素网格对齐、清晰对比度**，便于在 Unity 中作为 Sprite 使用。

**画布与透明规则**：有效元素以外的区域一律使用**透明（Alpha = 0）**；图片宽×高为**有效区域的外接矩形**（宽度 = 有效区域最大宽度，高度 = 有效区域最大高度），仅保留必要的安全边距（如九宫格边、图集防裁切边）。例如有效元素为圆形时，画布边长即为该圆直径（或直径 + 约定安全边）。

---

## 1. 弹窗底板背景（可九宫格切片）

**用途**：弹窗主面板背景，建议做 9-Slice 以适配不同尺寸。

**Prompt：**
```
像素风弹窗底板背景，用于九宫格切片 UI。有效区域为柔和圆角矩形，限色 8～16 色：暖棕/米色底、深棕色描边（2～3 像素轮廓），内侧轻微阴影、上缘一点高光。无渐变，纯块面色、像素边缘清晰。风格：休闲游戏/弹弹堂，干净易读。画布尺寸 = 有效底板的外接矩形（含四边约 16 像素九宫格安全边），即整体约 400×300 或 512×384 像素。有效区域外一律透明（Alpha=0）。PNG。
```

**尺寸建议**：画布宽×高 = 有效底板外接矩形 + 九宫格边距（四边约 16px），如 400×300 或 512×384。

---

## 2. 关闭按钮（默认态）

**用途**：弹窗右上角关闭按钮，未按下状态。

**Prompt：**
```
像素风关闭按钮图标，8～16 色。有效元素：深红或深棕的圆形/圆角方形底，上绘白色或浅灰「X」，2 像素描边/边框，像素网格对齐，无抗锯齿。画布尺寸 = 有效按钮的外接矩形（若为圆形则宽高 = 直径；可加约 4 像素安全边防图集裁切）。有效区域外一律透明（Alpha=0）。PNG。
```

**尺寸**：画布 = 有效圆形/圆角方形的外接矩形（例如直径 56px 则 56×56，或 64×64 含安全边）；可再提供 32×32 小尺寸版本。

---

## 3. 关闭按钮（悬停/按下态）

**用途**：关闭按钮 hover 或 pressed 状态，可与默认态同尺寸做状态切换。

**Prompt：**
```
像素风关闭按钮图标按下/悬停态，与默认态同风格。有效元素：背景略加深的圆形/圆角方 + 「X」向右下偏移 1～2 像素以表现按下感。画布尺寸 = 有效按钮的外接矩形（与默认态一致，有效区域外透明）。同套限色与像素网格。PNG，有效区域外 Alpha=0。
```

---

## 4. 确认按钮（主按钮）

**用途**：弹窗内“确定”“确认”等主操作按钮。

**Prompt：**
```
像素风确认/确定按钮。有效元素：绿色或金色圆角矩形按钮，2 像素深色描边，上缘轻微高光（1～2 像素线），无渐变、块面色。画布尺寸 = 有效按钮的外接矩形（宽×高 = 按钮最大宽×最大高，可加约 4 像素安全边），如 128×48 或 64×64。有效区域外一律透明（Alpha=0）。风格：弹弹堂/休闲 2D，小尺寸可读。PNG。
```

**尺寸**：画布 = 按钮外形外接矩形（如 128×48 横条或 64×64 方钮），按实际布局二选一或各做一份。

---

## 5. 取消按钮（次要按钮）

**用途**：弹窗内“取消”“关闭”等次要操作。

**Prompt：**
```
像素风取消按钮，与确认按钮同尺寸。有效元素：灰色或中性棕圆角矩形按钮，2 像素深色描边，上缘高光与确认按钮一致。画布尺寸 = 有效按钮的外接矩形（与确认按钮一致）。有效区域外一律透明（Alpha=0）。PNG。
```

---

## 6. 弹窗标题栏装饰条

**用途**：弹窗顶部标题区域背景条或装饰带。

**Prompt：**
```
像素风弹窗标题栏装饰条，8～16 色。有效元素：横向条带，2～3 层块面色带（中间深、上缘略亮），可选两侧简单缎带端头，仅用像素块，线条 1～2 像素、网格对齐。画布尺寸 = 有效条带的外接矩形（宽×高 = 条带最大宽×最大高，如 384×48）。条带上下/两侧外一律透明（Alpha=0）。PNG。
```

**尺寸**：画布 = 条带外接矩形（宽度略小于底板如 384，高度约 48px），与底板风格一致。

---

## 7. 弹窗四角装饰（可选）

**用途**：弹窗四角的小装饰，增强弹弹堂/休闲风格。

**Prompt：**
```
像素风弹窗四角装饰，单角（如左上角）。有效元素：小缎带卷、木纹角撑或小宝石等，色板与弹窗底板一致（棕/金/米色），1～2 像素描边，像素精确、无平滑曲线。画布尺寸 = 有效装饰的外接矩形（例如装饰直径或边长约 24～28 像素则画布 32×32，或更小紧贴图案）。有效区域外一律透明（Alpha=0）。PNG。可提供四角各一或一张可旋转复用。
```

**尺寸**：画布 = 单角装饰的外接矩形（如 32×32 或更小），需四个角可旋转或做 4 张图。

---

## 8. 通用分割线（横条）

**用途**：弹窗内分隔标题与内容、或内容与按钮区域。

**Prompt：**
```
像素风 UI 横向分割线。有效元素：单像素或 2 像素粗的横线，深棕或灰色，可选轻微双线（阴影），直线、网格对齐。画布尺寸 = 有效线条的外接矩形（高 = 线粗或阴影范围如 8 像素，宽 = 线段长度如 256，可按需平铺）。线条上下外一律透明（Alpha=0）。PNG。
```

**尺寸**：画布 = 线条外接矩形（如 256×8），宽度可随内容区拉伸或平铺。

---

## 9. 弹窗遮罩/半透明蒙版（可选）

**用途**：弹窗后的全屏暗色蒙版，通常用 Unity 色块 + 透明度实现即可；若需贴图可用小图平铺。

**Prompt：**
```
像素风 UI 遮罩用轻微噪点或纹理，可平铺。有效区域为整张图，极深灰带轻微纹理，4～8 色，低对比度。画布尺寸 = 单块纹理外接矩形（如 64×64）。可选：轻微网格或点阵。PNG，半透明亦可；若为全图有效则无需“有效区域外透明”。
```

**说明**：多数项目用纯色 + Alpha 做遮罩，本项仅当需要纹理时使用；遮罩贴图通常整图有效，无“有效区域外”需透明。

---

## 使用与规范小结

| 资源           | 画布尺寸（= 有效区域外接矩形） | 格式 | 备注                     |
|----------------|--------------------------------|------|--------------------------|
| 弹窗底板       | 400×300+（含九宫格边约 16px）  | PNG  | 有效区域外透明           |
| 关闭按钮       | 直径×直径 或 64×64（含安全边） | PNG  | 默认 + 悬停/按下各一     |
| 确认/取消按钮  | 128×48 或 64×64                | PNG  | 与底板色系统一           |
| 标题栏条       | 384×48                         | PNG  | 与底板风格一致           |
| 角装饰         | 32×32 或更小紧贴图案           | PNG  | 四角或可旋转             |
| 分割线         | 256×8                          | PNG  | 可平铺                   |

- **有效区域外透明**：所有资源有效元素以外一律透明（Alpha=0），不填纯色或渐变底。
- **画布尺寸**：图片宽×高 = 有效区域外接矩形的最大宽×最大高，仅保留必要安全边距（九宫格边、图集防裁切边）；例如圆形按钮则画布边长 = 直径（或直径 + 安全边）。
- **色板**：整批资源尽量共用一套 8～32 色（如棕/米色/深棕/金/绿/灰），保证界面统一。
- **描边**：按钮与底板用 2px 左右深色描边，线条粗细统一。
- **风格**：弹弹堂/休闲 2D、像素块明确、避免写实材质与复杂渐变。

以上 prompt 可直接用于 AI 绘图或交给美术按规范出图；若引擎内已有主题色变量，可在 prompt 中替换为具体色值描述。
